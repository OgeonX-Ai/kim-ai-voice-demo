# Google Gemini API Key Setup

## Step 1: Google AI Studio
https://aistudio.google.com/

## Step 2: Create API key
Settings → API keys → Create key

## Step 3: Add to the app
Settings → AI Providers → Google Gemini  
Paste the API key.

## Recommended models
- gemini-1.5-pro
- gemini-1.5-flash

## Notes
Gemini keys are project-based. Usage limits apply.
