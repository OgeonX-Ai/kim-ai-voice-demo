# Anthropic (Claude) API Key Setup

## Step 1: Create account
https://console.anthropic.com/

## Step 2: Create API key
Dashboard → API Keys → Create key

## Step 3: Add to the app
Settings → AI Providers → Anthropic  
Paste API key and select Claude model.

## Recommended models
- claude-3-5-sonnet
- claude-3-opus

## Notes
Anthropic requires correct request formatting; the app handles this automatically.
