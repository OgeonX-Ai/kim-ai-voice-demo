# Mistral API Key Setup

## Step 1: Create account
https://console.mistral.ai/

## Step 2: Create API key
Dashboard → API Keys

## Step 3: Add to the app
Settings → AI Providers → Mistral  
Paste the API key.

## Recommended models
- mistral-large
- mistral-small

## Notes
Mistral uses OpenAI-compatible APIs.
