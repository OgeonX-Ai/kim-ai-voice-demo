Local setup

Start the enterprise-ai-gateway locally

Run the backend on localhost so the Call Whisper demo can reach /healthz and /v1/audio/transcribe-file.

Clone & install

git clone https://github.com/OgeonX-Ai/kim-ai-voice-demo.git
cd kim-ai-voice-demo/enterprise-ai-gateway
python -m venv .venv && source .venv/bin/activate
pip install -r requirements.txt # or npm install if using the Node stub

Run

uvicorn server:app --host 0.0.0.0 --port 8000

Prefer Node? Launch the Express stub with npm install then node server.js.

Verify

  • /healthz returns OK
  • /docs (if FastAPI) loads
  • /v1/audio/transcribe-file accepts multipart uploads

Common issues

  • Wrong venv: ensure you activated .venv
  • Port mismatch: page defaults to 8000
  • CORS blocked: enable CORS middleware in the gateway
  • Missing endpoint: add a Whisper/Azure handler at /v1/audio/transcribe-file
